Replenish order deposit with `id` by `amount`

## Usage

```
spctl orders replenish-deposit <id> <amount> [OPTIONS]
```

## Arguments

|**Name**|**Description**|
| :- | :- |
|`id`|Order `id`|
|`amount`|Amount of tokens|

## Options

|**Name, shorthand**|**Default**|**Description**|
| :- | :- | :- |
|`--config`|`./config.json`|Path to the configuration file|
