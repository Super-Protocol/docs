Display the balance of TEE and MATIC tokens in your wallet.

## Synopsis

```
./spctl tokens balance [option]
```

## Option

| **Name** |**Description**                      |
|:--------------------|:-------------------------------------|
| `--config`          | Path to the configuration file. Default is `./config.json`       |
