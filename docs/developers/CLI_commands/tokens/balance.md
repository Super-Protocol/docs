Get the balance of TEE and MATIC tokens in your wallet.

## Usage

Syntax.

```
./spctl tokens balance [OPTIONS]
```

## Options

| **Name, shorthand** | **Default**     | **Description**                      |
|:--------------------|:----------------|:-------------------------------------|
| `--config`          | `./config.json` | Path to the configuration file       |
