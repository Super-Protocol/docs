Display information on offer with `id` as it is stored in blockchain

## Usage

```
spctl offers get-info <type> <id>
```

## Arguments

|**Name**|**Description**|
| :- | :- |
|`type`|Offer `type` (choices: "tee", "value")|
|`id`|Offer `id`|
