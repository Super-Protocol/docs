Display detailed information on a value or compute offer with `id` as stored on blockchain.

## Usage

Syntax:

```
./spctl offers get-info <type> <id>
```

## Arguments

| **Name** | **Description**                 |
|:---------|:--------------------------------|
| `type`   | Type of offer: `tee` or `value` |
| `id`     | Offer ID                        |
