Cancel an order.

## Synopsis

```
./spctl orders cancel <orderId> [option]
```

## Argument

|**Name**|**Description**|
| :- | :- |
|`orderId`|Order ID|

## Options

|**Name**|**Description**|
| :- | :- |
|`--config`|Path to the configuration file. The default is `./config.json`|
