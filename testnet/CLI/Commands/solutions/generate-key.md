Generate solution signing key

## Usage

```
spctl solutions generate-key <outputPath> [OPTIONS]
```

## Arguments

|**Name**|**Description**|
| :- | :- |
|`outputPath`|Path to save a key file|

## Options

|**Name, shorthand**|**Default**|**Description**|
| :- | :- | :- |
|`--config`|`./config.json`|Path to the configuration file|
